# Gatepath Realtors

Premium land sales platform — built with TanStack Start, React 19, Supabase, and Cloudflare Workers.

## Stack
- TanStack Start (SSR) + TanStack Router
- React 19 + TailwindCSS v4
- Supabase (database, auth, realtime)
- Paystack (secure payments)
- Cloudflare Workers (edge deployment)

## Local Setup
Copy `.env.example` to `.dev.vars` and fill in your Supabase and Paystack keys, then run:
```
npm install
npm run dev
```
